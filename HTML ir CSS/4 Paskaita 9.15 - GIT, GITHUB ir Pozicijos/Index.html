Position

  • Static - Statinė
  • Relative - Reliatyvioji
  • Absolute - Absoliučioji
  • Fixed - Fiksuotoji
  • Sticky - Lipnioji

Naudojant bet kurią apart Statinės pozicijos, gauname prieigą prie 5 naujų stilių: top, left, right, bottom, z-index.
Šitie stiliai nurodo per kokį atstumą elementas turi būti atitrauktas.
Z-index rodo elemento prioritetą sluoksniose.

Terminalo Komandos

  • cd - change directory - pakeisti direktoriją - vaikščiojimas tarp aplankalų.
  • ls - list files and directories - išvardinti failus ir aplankalus.
  • CTRL C - atšaukia vykdoma veiksmą.
  • TAB užbaigia frazę arba pateikia pasiūlimus
  • clear - nukelia visą terminale rodomą informaciją į viršų.cle

Git Komandos

  • init - Initialize - sukuria git repozitoriją
  • status - peteikia informacija apie tavo failus esamoje repozitorijoje
  • add - pridėti/paruošti/stage'ina failus comit'inimui
  • commit - sukuria kodo versiją iš stage'intų failų
