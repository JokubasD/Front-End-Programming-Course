  • dd - Aprašamojo sarašo termino paaiškinimas

  • br - Eilutės lūžis/break

  • thead - Lentelės viršus
  • tbody - Lentelės kūnas

  • src - Šaltinis (nuoroda į failus)

  • type - Įvesties lauko tipas (jų yra daug)

  • class - Elemento klasė
  • style - naudojamas pakeisti teksto stilių
  • rel - naudojamas jungti kelis documentus vieną prie kito
