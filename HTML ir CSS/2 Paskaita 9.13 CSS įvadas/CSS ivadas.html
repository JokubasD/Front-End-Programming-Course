CSS Rašymo būdai

  • Inline - Linijinis rašomas tag'o viduje naudojant style atributą
  • Internal - Vidinis rašomas style tag'o viduje(head dalyje).
  • External - Išorinis rašomas atskirame CSS saile kurį reikia prijungti prie HTML naudojant link tag'ą

CSS Selector'iai

  • Element - Parenkamas elementas pagal jo pavadinimą (p{})
  • ID - Sukuriamas susigalvotas id pavadinimas ir priskiriamas vienam elementui (#ID{})
  • Class - Sukuriamas susigalvotas klasės pavadinimas ir priskiriamas bet kokiaam kiekiui elementų (.KLASĖ{})

CSS Stiliai

  • color - skirtas pakeisti teksto spalvą (color:red)
  • font-size - skirtas pakeisti teksto dydį (font-size: 20px)
  • background-color - skirtais pakeisti fono spalvą (background-color:aqua)
  • font-family - skirtas pakeisti teksto sriftą (font-family: monospace)
  • text-align - skirtas lygiuoti tekstą (text-align: center)
  • font-weight - skirtas pakeisti teksto storumą (font-weight: bold)
  • line-height - skirtais pakeisti teksto aukštį (line-height: 20px)
  • border - skirtas nurodyti rėmelio nustatymus (border: 3px solid blue)
  • border-radius - skirtas apvalinti rėmelio kampus (border-radius: 10px)
  • font-style - skirtas suteikti tekstui stilių (font-style: italic)

Spalvos

  • Spalvos vardo naudojimas. Yra 140 spalvų vardų. (red)
  • HSL naudojimas. Niekas jo nenaudoja. (hsl(0,100%, 50%))
  • HEX naudojimas. Naudojamas taip pat plačiai kaip ir RGB. (#FF0000)
  • RGB naudojimas. Naudojamas taip pat plačiai kaip ir HEX (rgb(255, 0, 0))

Permatomos spalvos

  • HSL permatomumas nurodomas nuo 0 iki 1. (hsla(0, 100%, 50%, 1))
  • HEX permatomumas nurodomas nuo 00(pilnai permatomas) iki FF(pilnai matomas). (#FF0000FF)
  • RGB permatomumas nurodomas nuo 0 iki 1. (rgba(255, 0, 0, 1)

HEX ir RGB

RGB - RedBlueGreen. Kiekvienos spalvos kiekio nurodai nuo 0 iki 255.

HEX - Kiekvienos spalvos kiekio nurodai nuo 00 iki FF
