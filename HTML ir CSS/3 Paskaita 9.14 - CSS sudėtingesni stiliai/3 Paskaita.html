Failų kelių nurodymai

Noridant failo vardą - nueinama į nurodytą failą.
Norint atlikti sekantį veiksmą - reikia parašyti "/".
Norint žengti žingsnį atgal iš aplankalo - reikia parašyti "../".

Puslapio struktūra

  • header - puslapio viršsus, kuriame norodoma informacija apie puslapį/menu.
  • nav - skirtas kurti puslapio menu
  • section - didžiulė puslapio dalis
  • div - maža puslapio dalis
  • main - pagrindinė puslapio dalis
  • article - main viduje esanti sekcija
  • aside - šoninė puslapio sekcija
  • footer - puslapio apačia, kurioje yra įvairi informacija kaip: terms and uses; copyrights; privacy policy; partners...

Menu Info

Kuriant menu - naudojamas nav tag'as, kurio viduje yra rikiuotas/nerikiuotas sąrašas(ol/ul). To sąrašo elementų (li) viduje yra nuorodos tag'as (a), kuris nurodo į kažkurią puslapio dalį arba kitą puslapį.

Nuorodos tag'o (a) atidaromoje dalyje esantis atributas href nurodo į kur vartotojas turi būti nukeltas paspaudos ant nuorodos.

Jeigu atributas nurodo į kažkokį ID (href="#ID_pavadinimas"), tuomet vartotojas yra nukeliamas į puslapio dalį, kuriai yra suteiktas nurodytas ID (id="ID_pavadinimas").

Jeigu href atributas nurodo į kitą html failą (href="./aplankalas/failas.html"), tuomet vartotojas yra nukeliamas į nurodytą html failą.

Inline, Block ir Inline-block

Inline

Inline elementai yra tokie elementai, kurie užima tik tiek vietos ekrane, kiek jiems yra reikalinga. Minusas tas, kad negalima keisti šių elementų dydžių. Pliusas tas, kad jie labai lengvai įtalpinami tarp kitų informacijos vienetų.
Tag'ų pavyzdžiai: a, span, img, input, button, label, select, textarea, cite...

Block

Block elementai yra tokie elementai, kurie užima visą leistiną plotį ir neįsileidžia kitų elementų į tą pačią eilutę. Minusas tas, kad užima visą leistiną plotį. Pliusas tas, kad galima keisti dydį.
Tag'ų pavyzdžiai: h1-h6, p, div, section, main, article, nav, ul, ol, blockquote, aside, table, canvas...

Inline-Block

Inline-Block elementai yra tokie elementai, kurie pasisavino geriausias Blokinių ir Linijinių elementų savybes. Jie užima tiek vietos kiek jiems reikia, bet jų dydį vis viena galima keisti.

Model Box

Margin - Atstumas nuo elemento iki aplinkinių elementų rėmelių (skirtingų elementų margin'ai persidengia ties viršum/apačia, ties šonais - ne).

+-- Border - Rėmelis, kuris yra tarp Margin ir Padding

Padding - Astumas nuo elemento content(turinio) iki elemento rėmelio

Content - Elemento turinys. (blokinio elemento turinys užims visą leidžiamą eilutę arba nurodytą dydį).
